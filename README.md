# IntelligentCareSystem_CV

## 1. cv_part
-  **face_collection**: 包含了义工信息的收录， 以及陌生人的识别。
- **face_extra_detect**: 包含了人脸的检测与提取。
- **emotion**: 包含了KNN, ANN,MINVGG对人脸的smile表情和neutral表情的分类训练。
